## Tools requires 

This project has a farrago of dependencies:

* A unix-system.
* Git LFS to download an archive.
* `tar` to unzip an archive.
* GDAL (`gdal_translate`) for fixing a corrupt image.
* ImageMagick (`magick`) for extracting images.
